# ArduinoAutomaticHandSanitizer
Arduino Project – Touch Free Automatic Hand Sanitizer

This project is designed to make a touch free automatic hand sanitizer with an ultrasonic sensor and a servo motor. or more information, visit https://eddragon.com/arduino-project-automatic-hand-sanitizer/.
